# JavaPhoto
It's just a simple Paint program made by me and my friend in 2017.

### __Functions__

- The kind of pen
- Erase the screen
- Pen thickness
- Pen color.

고1때, 친구와 둘이서 만든 단순한 그림판입니다.
펜 종류, 지우기, 펜의 굵기, 색상등을 구현해 그림을 그릴 수 있게 했습니다.
제가 맡은 부분은 GUI부분이였습니다.
